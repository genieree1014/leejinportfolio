Discovering Research Trends: Keyword-Based Publication Search and Wordcloud Analysis

Based on past experiences, whenever I start a new research topic or delve into a new subject, the researcher in me begins searching for publications to identify the latest trends. During this process, the number of publications I read increases, and I realized it is important to have an effective method for properly documenting the list of publications. In my opinion, using an Excel spreadsheet that includes basic publication information such as publication dates, author names, publication links, titles, and abstracts is a good starting point (instead of manually summarizing this information in Excel!). Additionally, a word cloud summarizing the latest trends based on the titles of these publications is also very helpful in helping me understand the topics. Therefore, I dedicated some time outside working hours to develop my first Shiny app containing these features using Python. I hope you enjoy using the app!
